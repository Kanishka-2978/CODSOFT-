# Python Programming Internship

This repository contains all the projects completed as part of the CODSOFT Python Programming Internship.

## Tasks Completed

### Task 1 – To-Do List
A console-based application to manage daily tasks by adding, viewing, completing, and deleting tasks.

### Task 2 – Simple Calculator
A basic calculator that performs addition, subtraction, multiplication, and division operations.

### Task 3 – Password Generator
A Python application that generates strong and secure random passwords based on user input.

### Task 4 – Rock Paper Scissors Game
A console-based game where the user plays Rock, Paper, Scissors against the computer.

### Task 5 – Contact Book
A contact management application that allows users to add, view, search, update, and delete contacts.

## Technologies Used

- Python 3
- Console-Based Applications
