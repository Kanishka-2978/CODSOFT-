Rock Paper Scissors Game

Description

This project is a Rock Paper Scissors Game developed using Python as part of the CODSOFT Python Programming Internship. The game allows the user to play against the computer through a simple console-based interface.

Features

- Play Rock, Paper, Scissors against the computer
- Random computer choice
- Score tracking
- Winner determination
- Input validation
- Menu-driven interface

Technologies Used

- Python 3

Learning Outcomes

- Python Functions
- Random Module
- Conditional Statements
- Loops
- User Input Validation
