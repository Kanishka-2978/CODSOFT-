Password Generator

Description

This project is a Password Generator developed using Python as part of the CODSOFT Python Programming Internship. It generates strong and secure passwords based on user preferences such as password length and character types.

Features

- Generate Random Passwords
- User-Defined Password Length
- Supports Uppercase Letters
- Supports Lowercase Letters
- Supports Numbers
- Supports Special Characters
- Input Validation
- User-Friendly Console Interface

Technologies Used

- Python 3

Learning Outcomes

- Python Functions
- Random Module
- String Module
- Loops
- Conditional Statements
- Input Validation
